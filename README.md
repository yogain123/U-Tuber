# U-Tuber

#### Using Youtube Data API v3 to get list of videos searched.

#### How To Use:

- git clone https://github.com/yogain123/U-Tuber
- Go inside the folder U-Tuber
- yarn install

#### How To Run Application:

- yarn start
- Navigate to http://localhost:3000/

#### How To Run Tests

- yarn test

#### How to build

- yarn build

#### Node Version

12.15.0

#### NPM Version

6.13.5

#### YARN Version

1.5.1
